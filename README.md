# DecodeLabs Internship Project 1

This repository contains my **Responsive CoffeeShop Website** built as part of the DecodeLabs Internship Program.  
The project demonstrates responsive frontend design using **HTML, CSS, and JavaScript**, following a mobile‑first approach.

---

## 📂 Project Structure
- `index.html` → Main webpage structure
- `style.css` → Styling with responsive design (Flexbox & Grid)
- `script.js` → Basic interactivity and logic
- `images/` → Assets used in the project

---

## 🚀 Features
- Mobile‑first responsive layout
- Semantic HTML5 structure for accessibility
- CSS Flexbox/Grid for adaptive design
- Fluid typography and clean UI
- Basic JavaScript interactivity
